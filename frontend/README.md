# Patient Monitoring Dashboard

React-based nurse dashboard for real-time patient monitoring.

## Quick Start

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm run dev
```

Dashboard will open at http://localhost:5173

## Build for Production

```bash
npm run build
```
